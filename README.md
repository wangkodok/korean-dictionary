# 국어 사전 검색 플랫폼

이 프로젝트는 일상생활에서 모르는 단어를 검색하고 저장, 삭제할 수 있는 검색 서비스를 만들었습니다.

어려운 기술적인 부분보다는 기초적인 부분에 충실하게 학습하여 구현을 해나가는 것을 목표로 삼았습니다.
